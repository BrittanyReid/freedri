Installing Node.js on Windows

Node.js is an open-source platform that allows you to create web servers using JavaScript, as well as use heaps of great modules through its package manager npm. In this tutorial, we’re going to run through how to install it on windows.

Installation

First, download and run the installer from nodejs.org. This will install both Node.js and npm.

Open command prompt as an administrator. To check if Node.js has been installed correctly, type:

node –v

Now, if you want to install any modules globally, you can use the following command. We’ll install express, a framework for node.js web applications.

npm install -g express

We’ll also want to install express-generator globally so we can use the express command to build our directory.

npm install -g express-generator

The node.js site has some recommendations on when to install globally or locally:

In general, the rule of thumb is:
  1. If you’re installing something that you want to use in your program, using require('whatever'), then install it locally, at the root of your project.
  2. If you’re installing something that you want to use in your shell, on the command line or something, install it globally, so that its binaries end up in your PATH environment variable.

Running Node.js

If you want to run a node.js server, follow these instructions. We’ll use express for this. Create a directory for your server, and navigate inside it. Create your directories using the following command:

express
Install all dependencies from the package.json file by entering: 

npm install
You can now run your web server using the command:

npm start
Open your web browser and go to http://localhost:3000/, or whatever port number is specified in your bin/www file. You should see a webpage like this.
